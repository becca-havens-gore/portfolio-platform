logo-horizontal.svg

Full lockup — mark + wordmark + subtitle. Embed in the portfolio header as an <img> or inline SVG. Inline SVG inherits currentColor from the parent element's color CSS property for automatic dark-mode adaptation.

logo-mark.svg

Mark only — use as a favicon (favicon.svg), social avatar, or anywhere space is constrained. At 16–20px the orange diamond still reads clearly against both light and dark backgrounds.

Inline SVG usage (recommended)

Paste the SVG code directly into your HTML so currentColor inherits from the header's text color. No separate file request, and the theme switcher handles dark/light automatically with zero extra CSS.

High-contrast variant

For the [data-theme="high-contrast"] override, change the bracket fill from currentColor to #FFFFFF and the diamond from #FF6B2B to #FFD700 — matching the HC accent already in your system.
